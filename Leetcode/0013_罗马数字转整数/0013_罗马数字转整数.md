# 0013_罗马数字转整数

> difficulty: easy

## 解法一：
首先，设想简单的情况，如果罗马数字中字符和数字存在一一对应的关系，那么这个问题就变得非常轻松了，value用来记录，每当我们读取一个字母时，字符串中的指针就向前走，value的数值随之增加即可。
<br>
困难之处在于，有些情况需要读掉2个字符，value值只增加1次，但是此类情况并不多，我们将其独立出来，每当读到此类字符时，指针向前读2位，value值只增加一次，如此即可成功转化。
<br>
特殊情况一共6类：
| 字符 | 数字 |
| ---- | ---- |
| IV   | 4    |
| IX   | 9    |
| XL   | 40   |
| XC   | 90   |
| CD   | 400  |
| CM   | 900  |
